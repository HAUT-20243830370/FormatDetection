# 毕业论文格式检测脚本

## 环境配置说明

### 1. 安装 Python

首先确保电脑上已安装 Python 3.6 或更高版本。

**下载地址**：https://www.python.org/downloads/

安装时请勾选 "Add Python to PATH" 选项。

### 2. 安装依赖库

在命令行中运行以下命令安装所需的 Python 库：

```bash
pip install python-docx
```

### 3. 准备文档

将需要检测的 Word 文档（.docx 文件）放置在与 `main.py` 相同的目录下。

### 4. 运行检测脚本

有两种方式运行检测脚本：

**方式一：直接运行（自动检测当前目录下的第一个 .docx 文件）**

```bash
python main.py
```

脚本会自动查找当前目录下的第一个 .docx 文件并进行检测。

**方式二：指定文档路径**

```bash
python main.py 你的文档名.docx
```

或者使用绝对路径：

```bash
python main.py "D:\path\to\your\document.docx"
```

---

## 检测功能说明

本脚本包含以下 9 个格式检测功能：

| 检测项目 | 说明 |
|---------|------|
| 图序编号检测 | 检查图序编号是否按章节从1开始，顺序正确 |
| 参考文献格式检测 | 检查GB/T 7714—2015格式和引用顺序 |
| 段落首行缩进检测 | 检查是否为2.0字符首行缩进 |
| 中文字体检测 | 检查中文字体是否为宋体（跳过表格） |
| 目录字体检测 | 检查目录中文字体是否为宋体，英文是否为Times New Roman |
| 关键词格式检测 | 检查"关键词"是否为黑体、不加粗 |
| 段后空行检测 | 检查段后空行（跳过分页符和表格） |
| 参考文献引用上标检测 | 检查正文中的[1]、[2]等是否为上标格式 |
| 段落段后间距检测 | 检查段落段后间距是否为0.0行/0.0磅 |

## 常见问题

**Q: 运行时提示 "ModuleNotFoundError: No module named 'docx'"**  
A: 请运行 `pip install python-docx` 安装依赖库。

**Q: 如何检测其他文档？**  
A: 使用命令行参数指定文档路径：`python main.py 你的文档名.docx`

**Q: 检测结果中出现了问题怎么办？**  
A: 查看检测输出的详细信息，根据提示修改文档格式即可。
